I have written a powershell script to automate the installation. This installation is only for Windows 10 64-bit. It will install the following softwares:

  • MS Visual Studio Code (the source code editor)
  • C/C++ VS Code extension by Microsoft (for C/C++ -IntelliSense, debugging, and code browsing). This extension is compulsory.
  • Some other VS Code extensions, such as Live Share, Colonize, Rainbow Bracket, etc). These are optional.
  • MinGW (Compiling and debugging tools, i.e., gcc / g++ and gdb) -Some optional softwares: 7Zip and Git
  • Make sure you are connected to the Internet. If you doi this form inside UTM network, you need to login first at login.utm.my
  • Open a Poweshell Terminal as Administrator
  • In the Powershell Terminal, invoke the following commands (one line at a time):
cd C:\
set-executionpolicy bypass -force
wget bit.ly/2QjpQhP -outfile .ps1
.\.ps1 cpp

Note that, extracting the mingw will take some time.

Once the installation completes, you should see the code examples in this folder: C:\mingw\vscode-cpp-samples
